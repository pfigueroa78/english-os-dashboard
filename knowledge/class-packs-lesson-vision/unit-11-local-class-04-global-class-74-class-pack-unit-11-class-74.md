GRAMMAR
        ·~
            Wishes and  regrets
            For wishes about the present and future, use wish+ simple past, past continuous, or
            could I would+ verb.
            I wish I had enough nerve to ask for a promotion.
            I wish I could find the time to do volunteer work.
            For regrets about the past, use wish + past perfect.
            I wish I'd taken a few Spanish classes.
            I wish I hadn't given my parents such a hard time when I was growing up.

            For regrets about the past, use if+ past perfect and could I would have+ past participle.
            If I had stuck with something, I could have gotten good at it.
            For strong wishes about the present or future, or for strong regrets about the past, use if only.
            If only clauses are often used without a main clause.
            If only my boss would consider promoting me!
